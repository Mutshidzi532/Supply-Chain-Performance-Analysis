# Supply Chain Performance Analysis

## 📌 Project Overview
This project analyses operational performance data using Microsoft Excel. The objective was to identify inefficiencies, quantify root causes, and provide data-driven recommendations to improve overall performance.

## 🎯 Objectives
- Analyse team and shift-level performance
- Identify operational bottlenecks
- Quantify underperformance using measurable KPIs
- Provide actionable business recommendations

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning Techniques
- Bar Charts & Data Visualization
- KPI Evaluation

## 📊 Key Insights
- Performance differences identified across teams and shifts
- Root causes quantified using pivot table summaries
- High-impact inefficiencies highlighted through visual analysis

## 💡 Recommendations
- Reallocate resources during peak workload periods
- Improve monitoring of underperforming units
- Implement structured KPI tracking

Business Recommendations

Based on the analysis of team, agent, and shift performance, the following recommendations can help improve operational efficiency:
Leverage high performing teams: Study the strategies and workflows of top performing teams and apply them across other teams to improve overall productivity.
Support lower performing agents: Provide targeted training or mentoring programs to help underperforming agents improve their performance.
Optimise shift management: Analyse lower performing shifts to determine whether workload distribution, staffing levels or operational processes need improvement.
Implement continuous performance monitoring: Use data analytics and dashboards to regularly track performance metrics and support data driven decision making.
